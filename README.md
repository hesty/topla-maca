## Run App

Clone the repo

```bash
  git clone https://github.com/hesty/topla-maca.git
```

Go to the project directory

```bash
  cd topla-maca
```

Install dependencies

```bash
  flutter pub get
```

Run simulator or emulator

- IOS
```bash
  open -a simulator 
```
- Android

```bash
  emulator -list-avds
  emulator @{AVD NAME}
```

Run the app

```bash
  flutter run
```
